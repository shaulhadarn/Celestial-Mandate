A deployable structure that extracts bonus resources based on planet type. Built from the colony menu, visible on the planet surface, and repositionable by drone during exploration.

5. Planet Exploration — Colony Visible + Drone Interaction

Three changes to exploration mode

A. Colony Visible
Colony buildings group set to visible = true. Hub + all buildings rendered on terrain. Drone starts near the colony hub instead of hardcoded (25, 25).
B. Harvesters on Surface
Each built harvester rendered as a distinct 3D structure (drilling rig / extractor look) at its stored position on the terrain. Glowing beacon light so it's easy to spot.
C. Drone Interaction
When drone flies near a harvester (within ~15 units), a floating "Relocate" button appears. Activating it enters placement mode — fly to new spot, confirm with button press. Harvester moves.
